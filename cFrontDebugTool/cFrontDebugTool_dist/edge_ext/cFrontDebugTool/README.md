# 测试环境debug效率工具 - 尝试用ai编程助手协作完成开发

## 支持范围
平台和业务的测试环境，包括星瀚、星空旗舰等（通过环境域名判断），不支持客户环境，因为会泄漏debug相关信息
## 具体功能点
    1. 新增chrome插件，提供功能面板或下拉菜单 -ok
    2. 支持点击菜单按钮一键进入debug模式 -ok
    3. 支持一键打开旧开发平台、新开发平台 -ok 
    4. 支持一键在一个当前测试环境挂另一个测试环境前端代码（比如sit挂dev-ok、后端本地环境挂前端开发本地环境-?、测试环境挂前端开发本地环境-ok）
    5. 一键打开某表单的设计器（可用户录入formId） -？【暂无方案】
    6. 可配置前端自己本地开发环境的代码，一键在当前测试环境挂接自己本地开发环境到当前测试环境 -ok
    7. 一键打开某单据（需要录入formId）-ok
    8. 一键打开各测试环境：dev、sit、bos_baseline_a、baseline_a -ok
    9.一键打开移动端单据（url访问格式与pc不同）,默认带上当前页面上选中的formId -50%
    9. ？？其他可协助业务开发提高效率的功能
