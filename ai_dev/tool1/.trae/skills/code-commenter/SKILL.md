# 代码注释生成器

## 功能描述

该技能用于为代码添加函数级注释，确保代码的可读性和可维护性。

## 使用场景

- 当用户需要为现有代码添加注释时
- 当在生成代码时需要添加函数级注释时
- 当需要提高代码可读性和可维护性时

## 操作指南

1. 分析需要添加注释的代码文件
2. 识别其中的函数定义
3. 为每个函数添加详细的函数级注释
4. 确保注释包含：
   - 函数的功能描述
   - 参数说明（如果有）
   - 返回值说明（如果有）
   - 可能的异常或注意事项（如果有）

## 示例

### 输入：
```javascript
function calculateSum(a, b) {
  return a + b;
}
```

### 输出：
```javascript
/**
 * 计算两个数的和
 * @param {number} a - 第一个加数
 * @param {number} b - 第二个加数
 * @returns {number} 两个数的和
 */
function calculateSum(a, b) {
  return a + b;
}
```

## 支持的语言

- JavaScript/TypeScript
- Python
- Java
- C/C++
- PHP
- 其他常见编程语言

## 注意事项

- 保持注释的简洁明了
- 确保注释与代码逻辑一致
- 避免过度注释，只注释必要的信息
- 使用中文编写注释内容，除非代码本身是英文文档
