# 🌈 Rainbow Star Adventure (彩虹星星大冒险)

一个有趣的跨平台游戏，支持桌面版（Python/Pygame Zero）和Web版（HTML5/JavaScript）。

## � 项目背景

本项目基于 **AI Agent** 技术快速生成，专为儿童设计的休闲小游戏。通过人工智能辅助开发，从概念到实现仅需极短时间，展示了AI在游戏开发领域的强大能力。游戏采用简洁直观的操作方式，色彩丰富的视觉设计，以及轻松愉快的音效，非常适合儿童娱乐和亲子互动。

## �🎮 游戏介绍

在这个游戏中，你需要控制一个可爱的金色角色收集彩虹星星，同时避开紫色的障碍物。游戏有60秒的时间限制，看看你能获得多少分！

### 游戏特色

- 🎯 **简单有趣的玩法**：使用方向键或触摸控制角色移动
- 🌟 **彩虹星星**：收集各种颜色的星星获得分数
- 👾 **动态障碍物**：紫色敌人会在场景中移动，增加挑战性
- 🎨 **多彩背景**：每次收集星星都会随机更换背景颜色
- 🔊 **音效系统**：吃到星星和游戏结束时都有独特的音效
- 📱 **跨平台支持**：支持桌面端和Web端

## 🚀 快速开始

### 桌面版（Python）

#### 环境要求

- Python 3.7+
- Pygame Zero

#### 安装依赖

```bash
pip install -r requirements.txt
```

#### 运行游戏

```bash
# 方式1：使用启动脚本
./run_game.sh

# 方式2：直接运行
python3 game.py
```

#### 控制方式

- **方向键**：移动角色
- **空格键**：开始/暂停/重新开始游戏
- **鼠标**：点击按钮控制游戏

### Web版

#### 本地运行

```bash
cd web
python3 -m http.server 8000
```

然后在浏览器中访问 `http://localhost:8000`

#### 控制方式

- **方向键**：移动角色
- **触摸/滑动**：在移动设备上控制角色
- **点击屏幕**：开始/暂停/重新开始游戏

## 📁 项目结构

```
rainbow_star_adventure/
├── game.py              # 桌面版游戏主程序（Pygame Zero）
├── run_game.sh          # 桌面版启动脚本
├── requirements.txt     # Python依赖
├── rainbow_star.spec    # PyInstaller打包配置
├── web/

│   └── sounds/          # 音效文件目录
├── build/               # 构建输出目录

└── RainbowStarAdventure.dmg  # macOS安装包
```

## 🛠️ 技术栈

### 桌面版

- **Pygame Zero**：游戏框架
- **Python 3**：编程语言
- **PyInstaller**：打包工具

### Web版

- **HTML5 Canvas**：游戏渲染
- **JavaScript**：游戏逻辑
- **Web Audio API**：音效生成

## 🎨 游戏元素

### 玩家角色

- 金色圆形角色，带有动态嘴巴动画
- 使用方向键或触摸控制移动

### 星星

- 五角星形状，随机颜色
- 收集后获得1分
- 收集后会立即生成新的星星

### 障碍物

- 紫色圆形敌人，带有红色眼睛
- 在场景中自动移动
- 碰到会立即结束游戏

### 背景颜色

- 粉色、浅蓝、浅绿、浅黄、浅紫
- 每次收集星星后随机更换

## 🔧 开发说明

### 添加新功能

1. **修改游戏逻辑**：编辑 `game.py`（桌面版）或 `web/index.html`（Web版）
2. **添加音效**：将音频文件放入 `web/sounds/` 目录
3. **调整游戏参数**：修改游戏常量（如速度、数量等）

### 打包桌面版

```bash
pyinstaller rainbow_star.spec
```

打包后的应用将在 `dist/` 目录中生成。
